# HR Analytics Dashboard

## Overview

This project is an interactive HR Analytics Dashboard built using Power BI.
The dashboard analyzes employee attrition, hiring trends, salary distribution, overtime impact, and departmental insights.

## Key Insights

* 2021 recorded the highest hiring activity with 171 employees hired.
* Employee departures peaked in 2021 with 63 employees leaving.
* 127 employees who worked overtime left the organization.
* Research & Development had the highest attrition count.

## Dashboard Features

* Employee overview KPIs
* Attrition analysis
* Salary analysis by job role
* Hiring and departure trends
* Overtime impact visualization
* Interactive filters for Gender and Year

## Tools Used

* Power BI
* Power Query
* DAX
* Data Visualization
